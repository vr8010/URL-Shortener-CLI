URL Shortener CLI

A simple and lightweight Command Line URL Shortener that converts long URLs into short, shareable links directly from the terminal.

This project is designed for developers who want a fast and minimal way to shorten URLs without opening a browser.

🚀 Features
🔗 Shorten long URLs instantly
💻 Fully CLI-based tool
⚡ Fast and lightweight
📦 Easy to run and modify
🧑‍💻 Beginner-friendly project

URL shorteners convert long web links into shorter, easier-to-share links which are commonly used in messaging, social media, and development tools.

🛠️ Tech Stack
Python
CLI / Terminal Interface
API integration for URL shortening

```
URL-Shortener-CLI
```

⚙️ Installation

Clone the repository:
```
git clone https://github.com/vr8010/URL-Shortener-CLI.git
cd URL-Shortener-CLI
```
Install dependencies:
```
pip install -r requirements.txt
```
▶️ Usage

Run the CLI tool:
```
python main.py
```
Enter the URL you want to shorten:
```
Enter URL: https://example.com/very/long/url
```
Output:
```
Short URL: https://tinyurl.com/abc123
```
💡 Example
Input:
```
https://www.google.com/search?q=cybersecurity+tools
```
Output:
```
https://tinyurl.com/xyz123
```
